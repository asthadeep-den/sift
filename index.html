Live Wire — search the open market

Run a real web search for current EOIs, RFPs and tenders across the skilling funders. Results come back parsed: authority, type, deadline, source link, and a one-line relevance read tuned to this firm's profile.

Where it runs
Live search calls Claude's API with web search enabled. Inside the Claude artifact view it works as-is. On your deployed (Vercel) build, flip LIVE.mode to "proxy" in the script and drop in the search.js serverless function — a browser can't hold your API key directly.
Sift v2 · live-sourcing. Funder Feed links point to live official procurement pages or the funder's designated channel (GIZ → TendersOnTime). Live Wire runs a real-time web search and parses what it finds; treat every result as a lead to verify on the source page, not a confirmed live tender. Deadlines and eligibility shown are only as accurate as the public listing.
Scoring is a triage aid built on this firm's stated profile (₹100 Cr+ revenue, negative net worth, consortium-enabled). It is not eligibility certification — the Notice Inviting Tender always governs, and net-worth, solvency and prior-experience clauses are non-negotiable once stated. Tab / Shift-Tab to move.
